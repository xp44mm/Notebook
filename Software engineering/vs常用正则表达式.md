# vs常用正则表达式

选项为区分大小写：

因为正则表达式的点表示除`\n`以外的字符，即`[^\n]`，点号的匹配确实包括`\r`。
```
^11\.\d+ [^\r\n]+
^(Problem|Solution|Discussion|See Also)
^(WARNING|TIP)

替换为：
\r\n### $0\r\n
```

删除行尾空格

增加段落空行

```
([.:?]\r\n)([A-Z]) #区分大小写，不选全字符匹配
$1\r\n$2
```

折叠多个空行为一个空行

```
(\r\n){3,} #不选择全字符匹配
\r\n\r\n
```

替换后单引号’为英文单引号
